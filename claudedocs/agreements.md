# Agreements / Notes

This file records lightweight team agreements that are **not** always-on rules.

## PRD/overview-first (no separate 1-pager by default)

- We develop using long-form `overview.md` / PRD as the **single source of truth (SST)**.
- We do **not** create a separate “1-pager” document by default.
- If a short summary becomes necessary, we add a **TL;DR (1-pager summary) section at the top of the PRD/overview**.
  - Template: `../templates/prd_tldr_header.md`
  - Triggers: PRD gets too long, scope drift debates repeat, frequent PRD changes need a stable summary at the top.
